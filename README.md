📚 DSA with Java

A comprehensive collection of Data Structures and Algorithms (DSA) problems solved using Java. This repository is created to strengthen problem-solving skills, prepare for coding interviews, and build a solid foundation in algorithms and data structures.

🚀 Features

- Java-based DSA implementations
- Beginner to Intermediate problems
- Well-structured and readable code
- Optimized solutions with explanations

📂 Topics Covered

- Variables & Data Types
- Operators
- Conditional Statements
- Loops
- Functions & Methods
- Arrays
- Strings
- Recursion
- Sorting Algorithms
- Searching Algorithms
- Bit Manipulation
- Linked List
- Stack
- Queue
- Trees
- Binary Search Tree (BST)
- Heap
- Hashing
- Graphs
- Greedy Algorithms
- Dynamic Programming
- Backtracking

🛠️ Tech Stack

- Java
- VS Code 
- Git
- GitHub

🎯 Goals

- Improve problem-solving skills
- Master core DSA concepts
- Prepare for coding interviews
